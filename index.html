Как пользоваться?

  • загрузите изображение через форму слева;
  • для получения координат x,y объекта кликните курсором мышки на нужную точку изображения (выделится кругом);
  • двигайтесь по часовой стрелке, щелкая курсором, где необходимо;
  • для сохранения координат нажмите на кнопку "Завершить" ниже;
  • для копирования строки с координатами кликните на иконку копирования под строкой;
  • для сброса всех координат нажмите на кнопку "Обновить";
  • для загрузки другого изображения кликните по кнопке "Загрузить другое изображение".

Координаты объекта:

Например,
Точка 1: x = 323, y = 430
Точка 2: x = 539, y = 430

Строка координат (в формате x,y):

Например, 323,430,539,430
